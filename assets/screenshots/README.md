# 截图目录说明

本目录用于存放项目文档中的截图素材。

## 子目录

- `web/`：Web 端截图
- `ios/`：iOS 端截图
- `android/`：Android 端截图
- `desktop/`：桌面端截图
- `api/`：API 开发相关截图
- `senior/`：中老年指南截图
- `make-money/`：变现指南截图

## 提交规范

- 推荐格式：`PNG`
- 命名示例：`01-home-and-new-chat.png`
- 提交前请检查：
  - 是否包含敏感信息
  - 是否与文档占位符一致
  - 是否在 PR 描述中说明新增了哪些截图

## 网页优化建议（新增）

- 统一截图比例（建议 16:9 或 9:16）
- 控制单图体积（建议小于 500KB）
- 关键截图优先准备：
  - Web 首页
  - iOS/Android 首次上手流程
  - API 调用成功回显
  - 中老年模式示例
