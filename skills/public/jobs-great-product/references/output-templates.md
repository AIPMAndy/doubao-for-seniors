# 输出模板

## 模板 1：One-Page Product Brief

```markdown
# 产品一页纸

## 1. Mission
[一句话使命]

## 2. Primary User
[核心用户画像]

## 3. Core Scenario
[高频场景]

## 4. Pain Point
[用户痛点句]

## 5. Product Thesis
For [user] in [scenario], this product helps [core job]
better than [current option] because [unique advantage].

## 6. Non-Negotiables
- [体验底线 1]
- [体验底线 2]
- [体验底线 3]

## 7. Success Signal
- [上线后 7 天可观测指标]
```

## 模板 5：Open Source Action Checklist

```markdown
- [ ] README：定位、截图、快速开始
- [ ] docs/：完整使用文档
- [ ] LICENSE：明确开源协议
- [ ] CONTRIBUTING.md：贡献流程
- [ ] ROADMAP.md：近期计划
- [ ] ISSUE_TEMPLATE：问题反馈模板
- [ ] CHANGELOG.md：版本更新记录
```
