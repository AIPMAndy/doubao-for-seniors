# 豆包 Prompt 库（开源可复制版）

> 适配本项目的中老年学习场景 + 家人协助场景 + 日常办公场景。  
> 使用方式：复制模板 → 替换方括号内容 → 不满意就用“追问模板”继续迭代。

## 1. 万能提问骨架（先记住这 1 套）

```text
背景：我现在在做[事情]。
目标：我希望得到[结果]。
约束：请用[字数/语气/格式]输出。
输出：请按[步骤/清单/表格]给我。
```

## 2. 中老年用户高频 Prompt（20 条）

### 2.1 生活与出行（5 条）

```text
1) 今天天气怎么样？请告诉我是否需要带伞和加衣。
2) 我想去[地点]，请给我地铁和打车两种路线。
3) 明天我要去医院复查，请给我出门前准备清单。
4) 请帮我写一条提醒家人的短信，礼貌、简短。
5) 请告诉我今天最该做的3件事，每条不超过15个字。
```

### 2.2 健康参考（5 条）

```text
1) 请用大白话解释“甘油三酯偏高”是什么意思。
2) 高血压饮食要注意什么？请给3条可执行建议。
3) 我最近睡眠不好，请给我今晚就能做的改善动作。
4) 这段体检报告请解释：哪些词需要去问医生。
5) 涉及用药请先提醒我：必须以医生意见为准。
```

### 2.3 写作与沟通（5 条）

```text
1) 帮我写一条生日祝福，给老同学，真诚一点。
2) 帮我写一条家庭群通知，提醒周末聚餐。
3) 请写一条请假说明，原因是家人住院，请假3天。
4) 请把下面这段话改得更礼貌、容易理解。
5) 请给我3种风格：正式版、温和版、幽默版。
```

### 2.4 看图识字与翻译（5 条）

```text
1) 请先识别图中文字，再告诉我重点3条。
2) 请把这句英文翻译成中文，并告诉我什么时候用。
3) 这是药盒图片，请告诉我“作用-用法-注意事项”。
4) 这是菜单，请推荐清淡少油的选择。
5) 看完图后，请给我“重点3条+风险1条”。
```

## 3. 家人协助 Prompt（10 条）

```text
1) 请把这段内容改写成适合60岁长辈看的版本，短句、少术语。
2) 请把这个操作步骤压缩成4步，每步不超过15字。
3) 请生成“教爸妈用豆包”的7天计划，每天10分钟。
4) 请给我一份“隐私三不发”提醒卡片文案。
5) 请把这个回答做成“可直接读给父母听”的口语版。
6) 请给我一个“长辈听不懂时”的追问模板。
7) 请把这条消息判断风险等级，并给核查步骤。
8) 请给我一个“每周复习清单”，适合家庭辅导。
9) 请输出家人协助记录表：日期/学会内容/卡点/下次计划。
10) 请生成一段鼓励话术，让长辈更愿意继续学。
```

## 4. Web/办公 Prompt（15 条）

### 4.1 信息总结

```text
1) 请先给结论，再给依据，最后给风险。
2) 请把这份材料做成“领导30秒读完版”。
3) 请输出“可验证信息”和“待核验信息”两栏。
4) 把这段内容转成“问题-原因-建议”的结构。
5) 请按优先级排序，告诉我先做哪3件事。
```

### 4.2 写作与汇报

```text
1) 请写一份周报：本周成果/风险/下周计划。
2) 请给我5个标题：专业版/口语版/故事版/数据版/争议版。
3) 请把这段文字改成公众号风格，1200字。
4) 请输出“简版（300字）+完整版（1200字）”。
5) 请把此方案改为PPT提纲（10页）。
```

### 4.3 会议与协作

```text
1) 请整理会议纪要：决策事项、待办、风险点。
2) 请按负责人和截止日期输出执行表。
3) 请把任务拆成今天可执行的6步。
4) 请给一个复盘模板：做得好/做得差/改进动作。
5) 请写一条跟进消息，礼貌但推进感强。
```

## 5. 进阶追问模板（10 条）

```text
1) 请再给一个更保守、更稳妥的版本。
2) 请指出你回答里最不确定的3个点。
3) 请把答案改成表格并加优先级。
4) 请把术语替换成普通人能懂的说法。
5) 请给一个“今天就能开始”的最小方案。
6) 请给一个“预算更低”的替代方案。
7) 请给一个“风险更低”的替代方案。
8) 请把这段改成老人群里可读版本。
9) 请补充一个反例，告诉我常见误区。
10) 请最后给我一份检查清单（checkbox格式）。
```

## 6. 安全与合规提示（建议固定追加）

把下面这句作为高风险场景固定尾注：

```text
请在回答结尾加一句：以上仅供参考，涉及医疗/法律/投资等问题请咨询专业人士。
```
