# 截图素材说明（可选）

> 本项目当前以“具体使用方法 + Prompt 模板”为主。
> 截图素材是可选增强项，不作为发布阻塞条件。

## 1. 当前策略

- 产品界面变化快，优先维护文本步骤和可复制 Prompt。
- 截图仅用于辅助理解，不作为核心信息来源。
- 当截图与最新界面不一致时，以文字步骤为准。

## 2. 什么时候建议补截图

- 新手反馈“找不到入口”时。
- 关键流程出现明显误解时。
- 某个版本界面长期稳定时。

## 3. 补图最低规范（如需）

- 存放目录：`assets/screenshots/`
- 建议格式：`PNG`
- 命名格式：`序号-场景名.png`
- 隐私处理：打码手机号、头像、会话隐私内容。

## 4. 推荐做法

- 优先补“方法文档”中的步骤说明。
- 再补“Prompt 模板”中的变量解释。
- 最后才补图，避免维护成本过高。

## 5. 替代方案（无截图也可读）

- 用“入口描述”替代截图（如：底部麦克风按钮）。
- 用“步骤编号 + 一句话动作”替代界面标注。
- 用“常见误区 + 纠正动作”替代视觉引导。

如果你希望后续做图文版，可在稳定版本再逐步补充。
