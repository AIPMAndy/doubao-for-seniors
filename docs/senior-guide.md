# 中老年用户使用指南（增强版）

> 面向 50 岁以上用户，用最简单语言实现“会用、敢用、常用”。

## 1. 先认识：豆包能帮你做什么

豆包就像一个随时在线的小帮手，可以帮你：

- 回答日常问题（天气、健康常识、生活技巧）
- 写日常文字（祝福、请假、通知）
- 看图识字（说明书、菜单、路牌）
- 翻译外语（旅游、购物常用句）
- 聊天解闷（讲笑话、讲故事）

## 2. 第一次使用（一步一步来）

### 2.1 下载安装

苹果手机：

1. 打开 `App Store`
2. 点“搜索”
3. 输入“豆包”
4. 点击“获取”安装

安卓手机：

1. 打开手机应用商店
2. 搜索“豆包”
3. 点击“安装”

电脑用户：

- 浏览器输入 `www.doubao.com`

## 3. 五个最实用场景

### 3.1 健康常识（仅参考）

你可以问：

- “高血压平时要注意什么？”
- “失眠怎么调整作息？”

⚠️ 重要提醒：涉及疾病诊断和用药，请一定咨询医生。

### 3.2 写短信和祝福

你可以说：

- “帮我写一条生日祝福，给老同学。”
- “帮我写一条通知，提醒明天聚餐。”

### 3.3 看图识字

1. 点击“图片”按钮
2. 选照片
3. 说“帮我读一下这张图写了什么”

> [截图占位 SENIOR-04] 图片上传按钮位置

### 3.4 外语翻译

- “这句英文是什么意思：Thank you very much”
- “帮我把‘我想买这个’翻译成英文”

### 3.5 聊天解闷

- “给我讲个笑话”
- “推荐几首老歌”

## 4. 三句万能提问模板

不会问时，直接照着说：

1. “我想做什么：___”
2. “请用简单的话告诉我”
3. “再说一遍，分 3 步告诉我”

## 4.1 高频可复制 Prompt（长辈版）

> 用法：先复制一句，再把城市、姓名、场景替换成自己的。

### A. 生活日常（5 条）

```text
1) 今天天气怎么样？请告诉我是否需要带伞。
2) 我想去医院，请给我地铁和打车两种路线。
3) 冰箱里有鸡蛋和青菜，给我两道简单菜，分步骤。
4) 帮我写一条提醒家人的短信，语气温和一点。
5) 请告诉我“明天要做的3件事”，每条不超过15个字。
```

### B. 健康参考（5 条）

```text
1) 请用简单话解释“血压偏高”是什么意思。
2) 高血压饮食注意什么？请给我3条可执行建议。
3) 我经常失眠，请给我今天就能做的作息建议。
4) 这段体检结果请帮我解释，哪些词需要问医生。
5) 涉及吃药问题，请先提醒我“以医生建议为准”。
```

### C. 看图识字与翻译（5 条）

```text
1) 请先读出图片中的文字，再讲重点。
2) 请把这段英文翻译成中文，并告诉我在哪些场景能用。
3) 这张药盒图片里，哪些内容最重要？
4) 这个菜单我看不懂，请推荐适合清淡饮食的菜。
5) 看完图片后，请给我“重点3条+风险1条”。
```

## 5. 常见问题

### Q1：会收费吗？

基础功能通常可免费使用，以页面说明为准。

### Q2：怕泄露隐私怎么办？

- 不说银行卡号、密码、身份证号。
- 不上传证件照片和隐私聊天记录。

### Q3：听不懂它的回答？

直接说：

```text
请用更简单的话，再说一遍。
```

### Q4：字太小看不清？

- 手机设置里调大字体。
- 让子女帮忙打开“显示放大”。

## 6. 给子女/家人的协助清单

1. 帮父母完成首次登录。
2. 现场示范 2 次语音提问。
3. 把“万能提问模板”写进便签。
4. 帮他们设置大字体和高音量。
5. 每周复习 1 次使用方法。

## 7. 网页落地建议（长辈专区）

网页里建议单独做“长辈模式”页面：

- 超大字体
- 一步一图
- 每步不超过一句话
- 固定“防骗提醒”提示框
