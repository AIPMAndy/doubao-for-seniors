# Android 端使用指南（增强版）

> 适用设备：Android 手机 / 平板  
> 目标：兼顾安装安全、使用效率和持续复用。

## 1. 官方入口

- 下载聚合页：`https://www.doubao.com/download`
- Android 入口（官方页面实时展示）：`https://www.doubao.com/android.apk`

建议优先从聚合页进入，避免安装旧包或非官方来源版本。

## 2. 安装与安全建议

1. 打开官方下载页，选择 Android 下载。
2. 若系统提示未知来源安装：仅在确认来源后临时开启。
3. 安装后立即关闭未知来源权限。
4. 登录账号并完成基础设置。

## 3. Android 端四类核心场景

### 3.1 日常问答

```text
请用“结论 + 依据 + 下一步”回答，不要超过 120 字。
```

### 3.2 学习与办公

```text
请把这段内容整理成：
1) 核心知识点
2) 记忆口诀
3) 复习清单
```

### 3.3 语音交互

```text
请先复述你理解到的需求，再给答案。
```

### 3.4 图像任务

```text
请识别图片中的关键信息，并指出我最容易忽略的风险点。
```

## 4. 提效动作（推荐）

- 建立“常用提问模板”备忘清单，减少重复输入。
- 针对复杂任务，固定使用两步法：
  1) 先要提纲
  2) 再要完整版本
- 每次输出后追问一句：

```text
如果我要明天就执行，请给我最小行动清单。
```

## 4.1 Android 可复制 Prompt（稳定优先）

### A. 稳定输出模板（4 条）

```text
1) 请按“结论-依据-行动”输出，不超过120字。
2) 请给我短版和长版两个答案。
3) 请先告诉我你理解的问题，再回答。
4) 输出时请使用编号列表，便于手机阅读。
```

### B. 学习办公模板（4 条）

```text
1) 把这段内容整理成“知识点-例子-练习题”。
2) 请给我会议纪要模板：决策/待办/风险。
3) 请把任务拆成今天能做的5步。
4) 请给我“明天继续执行”的接力清单。
```

### C. 追问修正模板（4 条）

```text
1) 请把刚才答案再简化一版，适合转发给家人。
2) 请再补充一个更稳妥、风险更低的方案。
3) 请告诉我刚才答案里最不确定的部分。
4) 请给我一个“容易踩坑”提醒清单。
```

## 5. 稳定性与排错

### 问题 1：输入后长时间无响应

- 检查网络切换（Wi-Fi / 4G / 5G）。
- 关闭并重开应用后重试。

### 问题 2：图片上传失败

- 检查相册权限。
- 压缩大图后再传。

### 问题 3：回答质量波动

- 增加上下文和格式要求。
- 指定身份与目标读者。
