# 🧓 豆包长辈版 - Doubao for Seniors

> 让爸妈今天就能开口用 AI

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](LICENSE)

## 🎯 这是什么？

一份专门为 50 岁以上用户设计的豆包使用指南。

**不是**教他们"什么是大模型"，**而是**让他们"今天就能用起来"。

## 😫 痛点

| 现状 | 本项目解决方案 |
|------|----------------|
| 教程太技术化，长辈看不懂 | 用最简单的话，一步一步教 |
| 功能太多，不知道从哪开始 | 只教最实用的 20%（语音提问、看图识字、翻译、写祝福） |
| 子女想教但不知道怎么教 | 提供"家人协助清单"，照着做就行 |
| 不敢问，怕问错 | 提供 18 条可复制话术，直接照着说 |

## 🚀 3 分钟上手

### 长辈自己学

1. 打开 [在线网站](site/index.html)
2. 点击"长辈模式"（大字更清晰）
3. 照着"常用话术"开口问

### 子女帮忙教

1. 阅读 [家人协助清单](docs/senior-guide.md#家人协助清单)
2. 帮长辈下载安装豆包
3. 演示一次语音提问
4. 把常用话术保存到长辈手机备忘录

## 💬 5 个最实用场景

### 1. 健康常识（仅参考）
> "高血压平时要注意什么？"

⚠️ 涉及疾病诊断和用药，请一定咨询医生。

### 2. 写祝福短信
> "帮我写一条生日祝福，给老同学，要温馨一点。"

### 3. 看图识字
> 拍照上传 → "帮我读一下这张图写了什么"

### 4. 外语翻译
> "帮我把'我想买这个'翻译成英文"

### 5. 聊天解闷
> "给我讲个笑话" / "推荐几首老歌"
