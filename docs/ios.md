# iOS 端使用指南（增强版）

> 适用设备：iPhone / iPad  
> 目标：在移动场景下把“随手问”升级为“可复用的效率系统”。

## 1. 核验信息（截至 2026-02-28）

- App 名称：`豆包 - 随时帮忙的 AI 助手`
- Bundle ID：`com.bot.doubao`
- iOS 最低版本：`iOS 15.0`
- 最近版本：`12.2.3`
- 最近版本发布时间：`2026-02-24`
- 查询接口：`https://itunes.apple.com/lookup?id=6459478672&country=cn`

## 2. 安装与首次配置

1. 打开 App Store，搜索“豆包”。
2. 核对应用信息后下载安装。
3. 首次打开完成登录。
4. 建议开启权限：
   - 麦克风：语音提问与语音交互
   - 相册/相机：拍照识图、图像任务
   - 通知（可选）：接收关键提醒

## 3. iOS 三种高效使用模式

### 3.1 快问快答模式（碎片时间）

场景：通勤、排队、临时查信息。

提示词模板：

```text
请用 3 句话回答：
1) 核心结论
2) 为什么
3) 我现在应该怎么做
```

### 3.2 任务处理模式（工作学习）

场景：写文案、做复盘、准备汇报。

提示词模板：

```text
你是我的[岗位]助理。
我需要在[时间]内完成[任务]。
请给我：
1) 任务分解清单
2) 每步输出模板
3) 风险提醒
```

### 3.3 图像理解模式（拍照即问）

场景：说明书、菜单、图表、课件。

提示词模板：

```text
请先识别图中关键信息，再输出：
1) 这是什么
2) 我最该关注的 3 点
3) 下一步建议
```

## 4. iOS 端“更像产品”的用法

- 固定 3 组常用模板：学习、工作、生活。
- 每天结束前让豆包生成“今日复盘 5 条”。
- 把高质量回答收藏到统一笔记库（如 Notion/备忘录）。

## 4.1 iOS 可复制 Prompt（移动场景）

### A. 通勤碎片场景（4 条）

```text
1) 请用30秒可读完的方式解释：___。
2) 我现在只有10分钟，请给我最小行动清单。
3) 请给我3个最重要结论，每条不超过20字。
4) 请先给结论，再给原因，再给下一步。
```

### B. 语音输入场景（4 条）

```text
1) 请先复述你听到的内容，再回答。
2) 如果你没听清，请先问我确认，不要直接猜。
3) 请把答案分成1、2、3三步说。
4) 请用口语化句子回复，避免专业术语。
```

### C. 图像理解场景（4 条）

```text
1) 请先识别图片文字，再给重点3条。
2) 这是说明书，请告诉我“用途-步骤-注意事项”。
3) 这是图表，请告诉我趋势和异常点。
4) 这是菜单，请推荐清淡、少油的选择。
```

## 5. 常见问题

### Q1：语音识别不准怎么办？

- 换安静环境或改用文本输入。
- 说短句并分段，不要一次性长句。

### Q2：输出太长看不完？

追问：

```text
请给我“30 秒读完版”，最多 5 条。
```

### Q3：担心流量和电量？

- 长任务建议在 Wi-Fi 下进行。
- 外出场景优先文本模式，通常更省电更稳定。
