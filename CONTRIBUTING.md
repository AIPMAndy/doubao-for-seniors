# 贡献指南 / Contributing

感谢你愿意帮助这个项目，让更多长辈更容易用上 AI。

## 1. 可以贡献什么

- 长辈真实使用案例（优先）
- 更好理解的提示词模板
- 教学流程优化（减少步骤、减少术语）
- 文档错别字、错误链接、Bug 修复
- 网站可读性/无障碍优化

## 2. 提交前检查清单

- 不包含身份证、手机号、验证码、银行卡等隐私信息
- 截图中不含聊天记录、联系方式等敏感内容
- 文案避免夸大医疗/法律/投资效果
- 涉及健康问题时保留“仅供参考，请咨询专业人士”
- 链接可点击、文件路径存在

## 3. 贡献流程

1. Fork 仓库并创建分支
2. 按最小改动原则提交修改
3. 在 PR 描述中写清楚：
   - 改了什么
   - 为什么改
   - 如何验证
4. 等待 review，按反馈迭代

## 4. 内容风格建议

- 多用短句，一步一句话
- 避免术语；必须出现时附一句解释
- 默认面向 50+ 用户阅读习惯
- 优先“能立刻复制使用”的内容

## 5. 截图提交说明

截图目录：

- `assets/screenshots/web/`
- `assets/screenshots/ios/`
- `assets/screenshots/android/`
- `assets/screenshots/desktop/`
- `assets/screenshots/api/`
- `assets/screenshots/senior/`
- `assets/screenshots/make-money/`

建议：

- 格式：PNG
- 命名：`01-what-this-screen-is.png`
- 单图体积建议小于 500 KB
