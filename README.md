## 🚀 3 分钟上手

### 长辈自己学

1. 下载"豆包" App（应用商店搜索）
2. 手机号注册登录
3. 点麦克风，说："今天天气怎么样？"
4. 搞定！就这么简单

### 子女帮忙教

1. 阅读 [家人协助手册](docs/family-guide.md)
2. 帮长辈装好、登录好
3. 演示一次语音提问
4. 把常用话术存到备忘录
5. 每周陪练一次

## 💬 5 个最实用场景

| 场景 | 话术示例 |
|------|----------|
| 问天气 | "今天天气怎么样？需要带伞吗？" |
| 写祝福 | "帮我写一条生日祝福，给老同学" |
| 看图识字 | 拍照 → "帮我读一下这张图" |
| 翻译 | "这句英文什么意思：Thank you" |
| 问健康 | "高血压饮食要注意什么？" |

⚠️ 健康问题仅供参考，一定要问医生！
