# 家人协助手册

> 手把手教你怎么教爸妈用豆包

---

## 📋 教学前准备清单

### 硬件准备
- [ ] 长辈的手机充满电
- [ ] 网络正常（WiFi 或流量）
- [ ] 手机存储空间充足（至少 500MB）

### 心理准备
- [ ] 预留 30 分钟不被打扰的时间
- [ ] 准备好耐心（可能要重复很多遍）
- [ ] 想好几个长辈感兴趣的话题

---

## 🎯 第一次教学（30分钟）

### 第1步：安装（5分钟）

**iPhone**：
1. 打开 App Store
2. 搜索"豆包"
3. 点"获取"
4. 等待安装完成

**安卓**：
1. 打开应用商店
2. 搜索"豆包"
3. 点"安装"

💡 **技巧**：提前帮他们装好，省得现场折腾。

### 第2步：注册登录（5分钟）

1. 打开豆包
2. 选择"手机号登录"
3. 输入长辈的手机号
4. 帮他们输入验证码
5. 登录成功

💡 **技巧**：把账号密码记在纸上，贴在手机壳里。

### 第3步：演示语音提问（10分钟）

**你先演示一遍**：
1. 点麦克风按钮
2. 说："今天天气怎么样？"
3. 等待回答
4. 说："看，就这么简单！"

**让长辈试一遍**：
1. 把手机给他们
2. 指着麦克风说："点这里"
3. 让他们问一个简单问题
4. 不管问什么，都要鼓励："问得很好！"

💡 **技巧**：第一次一定要让他们成功，建立信心。

### 第4步：保存常用话术（10分钟）

把这些话术存到长辈手机备忘录：

```
1. 今天天气怎么样？
2. 帮我写一条生日祝福
3. 这张图写的什么？（配合拍照）
4. 高血压要注意什么？
5. 给我讲个笑话
```

💡 **技巧**：设置成桌面快捷方式，一点就能看到。

## 📅 后续跟进计划

### 第1周：每天陪练5分钟
- 每天让他们问一个问题
- 不管问什么都鼓励
- 遇到问题记下来，下次解决

### 第2周：隔天陪练
- 开始让他们独立使用
- 遇到问题先让他们自己试
- 实在不行再帮忙

### 第3周起：每周检查一次
- 问问他们最近用了没
- 有没有遇到什么问题
- 教一个新功能（比如看图识字）

## 🚫 常见错误（避坑指南）

### ❌ 错误1：一次教太多
**问题**：恨不得把所有功能都教一遍
**后果**：长辈记不住，反而更害怕
**正确做法**：第一次只教语音提问，其他的以后再说

### ❌ 错误2：语气不耐烦
**问题**："这么简单都不会？"
**后果**：长辈自尊心受伤，再也不想学了
**正确做法**：多说"没关系，我再演示一遍"

### ❌ 错误3：用专业术语
**问题**："你要点这个 AI 对话框"
**后果**：长辈听不懂
**正确做法**："你点这个圆圆的按钮"

### ❌ 错误4：帮他们操作
**问题**：直接拿过手机帮他们弄
**后果**：他们永远学不会
**正确做法**：手把手教，但让他们自己点

### ❌ 错误5：期望太高
**问题**：希望他们一周就能熟练使用
**后果**：双方都有挫败感
**正确做法**：给3个月时间，慢慢来

## 💬 鼓励话术模板

### 他们成功时：
- "太棒了！你学得真快！"
- "看，你自己就会了！"
- "比我第一次用的时候强多了！"

### 他们失败时：
- "没关系，我们再试一次"
- "这个确实有点难，慢慢来"
- "你已经比昨天进步了"

### 他们想放弃时：
- "就差一点点了，再试试？"
- "你看，这个功能多方便，学会了以后..."
- "要不今天先到这，明天我们继续？"

## 📞 远程教学技巧

如果你不在长辈身边：

### 方法1：视频通话
1. 微信视频通话
2. 让他们把手机屏幕对着摄像头
3. 你看着屏幕指导

### 方法2：录制教学视频
1. 用自己的手机录屏
2. 边操作边解说
3. 发给长辈反复看

### 方法3：远程控制
1. 安装"向日葵远程控制"
2. 直接操作他们的手机
3. 边操作边语音解释
