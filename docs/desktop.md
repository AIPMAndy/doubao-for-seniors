# 桌面端使用指南（Windows / macOS，增强版）

> 目标：把桌面端从“聊天工具”升级为“日常生产力中枢”。

## 1. 适合谁

- 高强度写作、方案、复盘、代码任务用户。
- 需要多窗口、多文档并行处理的办公人群。
- 追求稳定输出和可复用模板的团队成员。

## 2. 官方入口

- 下载页：`https://www.doubao.com/download`

桌面端安装包与版本以下载页实时展示为准。

## 3. 桌面端三层工作方式

### 层 1：快速问答层（5 分钟内）

- 用于“解释概念、翻译短文、整理要点”。
- 产出要求：短、准、可复制。

### 层 2：任务推进层（30-60 分钟）

- 用于“写报告、做方案、整理会议纪要、生成 PRD 草稿”。
- 产出要求：结构化 + 行动清单。

### 层 3：项目协作层（长期）

- 用于“沉淀模板、统一文风、固定流程”。
- 产出要求：可标准化复用。

## 4. 推荐提示词策略

### 4.1 长文产出

```text
你是[角色]。
请先输出目录，再按目录逐节生成正文。
每节末尾加“可执行动作”。
```

### 4.2 翻译与改写

```text
请保留原意，翻译成[目标语言]，并提供：
1) 直译版
2) 本地化自然版
3) 专业术语对照表
```

### 4.3 信息检索整理

```text
请把问题拆成 3 个子问题分别回答，最后合并成 1 页结论。
```

### 4.4 代码与技术文档

```text
请给我：
1) 最小可运行示例
2) 关键参数解释
3) 常见报错排查表
```

## 4.5 桌面端 Prompt 模板库（深度工作）

### A. 长文与方案（4 条）

```text
1) 你是[角色]，请先出目录，再逐节写正文，每节给执行动作。
2) 请把这份方案改成“可汇报版”：背景、目标、路径、指标、风险。
3) 请输出“简版（300字）+完整版（1200字）”。
4) 请把该方案转成PPT大纲（10页）。
```

### B. 协作与管理（4 条）

```text
1) 请把以下任务拆成周计划，附负责人和截止日期。
2) 请生成项目周报：进展、阻塞、风险、下周动作。
3) 请设计一个复盘模板：做得好/做得差/下次改进。
4) 请输出一个“团队统一术语表”。
```

### C. 技术与文档（4 条）

```text
1) 请先给最小可运行代码，再给扩展版本。
2) 请输出接口文档模板：请求参数/响应示例/错误码。
3) 请给我“排错路径图”：先查什么，再查什么。
4) 请输出“上线前检查清单”，按高风险优先排序。
```

## 5. 团队协作建议

- 建立团队公共提示词库（按岗位分类）。
- 约定输出格式（标题层级、表格字段、结论句式）。
- 对关键对外文案设置人工复核流程。

## 6. 质量与风险控制

- 事实类输出：二次核验日期、数字、法规条款。
- 对外内容：统一品牌语气与术语。
- 技术内容：至少本地运行一次示例代码。

## 7. 网页落地建议（桌面端板块）

网页建议呈现：

1. 场景分层（快问 / 任务 / 协作）
2. 一键复制提示词
3. 质量清单
4. 团队模板下载入口
