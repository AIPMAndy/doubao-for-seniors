## 🎯 这是什么？

这是一个面向 **50 岁以上用户** 和 **家庭辅导者** 的豆包上手项目。

它不讲大模型概念，重点是让长辈今天就能开始用、并且敢持续用。

## 🚀 3 分钟上手

### 长辈自己学

1. 下载豆包 App（应用商店搜索“豆包”）
2. 手机号登录
3. 点麦克风说一句：“今天天气怎么样？”
4. 完成第一次可用

### 子女帮忙教

1. 先看 [家人协助手册](docs/family-guide.md)
2. 帮长辈完成安装和登录
3. 现场演示 1 次语音提问
4. 把常用话术放进备忘录
5. 每周复习 1 次

## 💬 5 个最实用场景

| 场景 | 话术示例 |
|------|----------|
| 问天气 | “今天天气怎么样？需要带伞吗？” |
| 写祝福 | “帮我写一条生日祝福，给老同学。” |
| 看图识字 | 拍照后问：“帮我读一下这张图。” |
| 翻译 | “这句英文什么意思：Thank you” |
| 问健康 | “高血压饮食要注意什么？” |

⚠️ 健康信息仅供参考，请以医生意见为准。
